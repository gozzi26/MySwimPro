mySwimPro

A wearable device for swimmers.

Create

Structure your workout and customize sets and exercises. Add optional warmup and cooldown exercises, set the distance and interval duration for each exercise.

Edit directly using our interface or download a template to import.

Each workout can be exported as a readable pdf or a text file.

Tempo

Each set has an optional tempo that can be set, with a built-in feature to test your tempo before you set it.

The device on the swimmer has a buzzer that will allow them to follow the set tempo as they swim.

Messages

As a coach you can interrupt your swimmer by sending a unique buzzer pattern to let them know to stop swimming.

Each swimmer's device has a force-sensitive pad to let them notify the coach.

Monitor

Our device can monitor heartrate, oxygen levels and activity periods.

After each workout you can view the graphed data.

Import Workout

To import a previous workout, upload the the text file that you have previously exported and it will automatically add the workout to the application.

Export Workout

Export your workout as a pdf to give your swimmers a physical copy. You will not be able to import the pdf version back into the application

Export your workout as a text file so you can later import it back into the application. This will make it so you won't have to go through the setup process again for this specific workout.
